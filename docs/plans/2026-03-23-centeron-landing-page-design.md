# CenterOn 랜딩페이지 디자인

## 개요

- **브랜드명**: 센터온(CenterOn)
- **목적**: 필라테스/요가/헬스 센터 원장님 대상 서비스 홍보 및 무료 체험 신청 유도
- **타겟**: 센터 원장님 (사업자)
- **CTA**: 무료 체험 신청 (이메일/연락처 수집)
- **위치**: 루트 `/` → 랜딩페이지, 기존 대시보드 `/dashboard` 유지
- **톤**: 에너지틱/역동적 — 강렬한 컬러 대비, 모션 풍부
- **기술**: Framer Motion (스크롤 기반 인터랙션, 선언적 애니메이션)

## 컬러 팔레트

| 이름 | HEX | 용도 |
|------|-----|------|
| Black | `#080708` | 다크 배경, 텍스트 |
| Crayola Blue | `#3772FF` | 주 포인트, CTA, 링크 |
| Scarlet Rush (Red) | `#DF2935` | 강조, 경고, 액센트 |
| Golden Pollen (Yellow) | `#FDCA40` | 서브 포인트, 하이라이트, 뱃지 |
| Alabaster Grey | `#E6E8E6` | 밝은 배경, 서브 텍스트 |

## 레이아웃 원칙

- 비대칭 그리드 (40/60, 60/40 등 불균형 배치)
- 섹션마다 레이아웃 방향 교차 (좌→우, 우→좌)
- 요소 기울어짐/겹침으로 위트있는 배치
- 대각선 구분선 또는 웨이브 디바이더로 섹션 전환
- 2025 수상작 특징 반영: 풀페이지, 대담한 타이포, 인터랙티브 모션

## 페이지 구조 (7섹션)

### 네비게이션 바
- 고정 상단, 스크롤 시 반투명 다크 + 블러
- 좌: CenterOn SVG 로고
- 우: 앵커 링크 (`기능` `리뷰` `시작하기`) + `로그인` 버튼
- 모바일: 햄버거 메뉴

### 섹션 1: Hero
- **배경**: Unsplash 필라테스/요가 이미지 풀스크린, 다크 오버레이 70%, Ken Burns 줌인, 2~3장 자동 페이드 전환 (5초)
- **텍스트 (좌측 하단 비대칭)**:
  - 서브: `필라테스 · 요가 · 헬스 센터 통합 관리` (Yellow, 작게)
  - 메인: **"센터 운영,\n이제 켜기만 하세요."** (White, 초대형 볼드, "켜기"에 Blue 하이라이트)
  - 설명: `강사 일정, 회원 관리, 급여 정산까지 하나로` (Grey)
- **CTA**: "30일 무료 체험" (Blue 배경), 아래 `신용카드 없이 바로 시작`
- **모션**: 텍스트 stagger 등장 (아래→위, 0.2초 간격), 스크롤 유도 화살표 바운스

### 섹션 2: Pain Point
- **배경**: Black `#080708`
- **카드 3개 비대칭 배치** (각각 기울어짐 rotate -2deg, 1deg, -1deg), Red 대각선 스트라이프 장식
- **내용**:
  1. "엑셀로 일정 관리, 아직도?" — 스케줄 겹침, 수기 입력 실수
  2. "정산할 때마다 야근" — 수업 횟수 세기, 계산 오류
  3. "회원이 늘수록 혼란도 늘고" — 담당 강사 변경, 연락처 찾기
- **하단**: `"이제 다른 방법이 있습니다"` (Yellow, 페이드인) → 웨이브 디바이더
- **모션**: 카드 회전하며 좌/우 교차 등장 (0.3초 stagger), hover 시 기울기 0도 + 스케일업

### 섹션 3: Features
- **배경**: Grey `#E6E8E6`
- **지그재그 배치** (홀수: 좌 텍스트 / 우 이미지, 짝수: 반대)
- **기능 4개**:
  1. "한눈에 보는 캘린더" (Blue 포인트) — 강사별 색상 수업 일정
  2. "자동 급여 정산" (Yellow 포인트) — 수업 횟수 × 단가 자동 계산
  3. "회원 · 강사 통합 관리" (Red 포인트) — 담당 강사 지정, 메모
  4. "강사도 직접 확인" (Blue 포인트) — 강사 계정으로 스케줄/정산 조회
- **스타일**: 세로 컬러 바, 반투명 대형 숫자 `01~04`, 기울어진 카드 위 스크린샷
- **모션**: 텍스트 좌/우 슬라이드인, 이미지 아래→위 + 회전, 숫자 팝인

### 섹션 4: Screenshot
- **배경**: Black → Blue 그라데이션 (대각선 135deg)
- **중앙 브라우저 목업** (3개 점 + 주소창, perspective rotateX 5deg)
- 안에 실제 대시보드 스크린샷
- **핫스팟 3개**: 클릭 시 말풍선 ("실시간 예약 확인", "운영 지표", "강사별 색상 구분")
- **플로팅 뱃지**: 좌 "다크모드 지원", 우 "모바일 반응형"
- **모션**: 목업 원근감 등장, 핫스팟 펄스, 뱃지 부유 애니메이션

### 섹션 5: Reviews
- **배경**: Grey `#E6E8E6`
- **타이틀**: "센터온과 함께하는 원장님들" (좌측 정렬) + Yellow 별점 장식
- **카드 3개 비대칭** (가운데 살짝 위로)
- **리뷰**:
  1. 김지현 원장 (필라테스) — "정산 3시간→10분" (Yellow 하이라이트)
  2. 박수진 원장 (요가) — "전화 문의 90% 감소" (Blue 하이라이트)
  3. 이동훈 원장 (헬스&PT) — "회원 200명+ 관리" (Red 하이라이트)
- **스타일**: White 카드, 큰 따옴표 장식, 아바타(Unsplash 인물) + 이름 + 센터 유형, 우상단 수치 뱃지
- **모션**: stagger 팝업, hover 들어올림, 수치 카운트업 애니메이션

### 섹션 6: CTA
- **배경**: Black `#080708` + Blue/Red 그라데이션 원형 블러 장식
- **좌측 60%**: `30일 무료, 부담 없이` (Yellow) + **"지금 센터를\n켜보세요."** (White 대형) + 체크리스트 (신용카드 불필요, 전 기능 무료, 1분이면 시작)
- **우측 40%**: 반투명 다크 카드 폼 — `센터명` `이름` `이메일` `전화번호` + "무료로 시작하기" (Yellow 배경, Black 텍스트)
- **모션**: 좌/우 슬라이드인, 배경 블러 floating, 제출 성공 시 축하 메시지 + 컨페티

### 섹션 7: Footer
- **배경**: Black `#080708` (CTA와 연결)
- **3컬럼 비대칭**:
  - 좌: CenterOn 로고 + `센터 운영을 켜다`
  - 중: 링크 (기능, 리뷰, 로그인, 이용약관, 개인정보처리방침)
  - 우: 연락처 (이메일, 전화)
- 상단 Yellow 얇은 구분선
- 하단: `© 2025 CenterOn. All rights reserved.`

## 필수 에셋

- **SVG 로고**: CenterOn 브랜드 로고
- **SVG 파비콘**: 로고 심볼 축약 버전
- **Unsplash 이미지**: Hero 배경 (필라테스/요가 2~3장), 리뷰 아바타 (3장)
- **UI 스크린샷**: 대시보드 페이지 캡처 또는 목업

## 기술 스택

- Next.js App Router (기존 프로젝트)
- Tailwind CSS 4 (기존)
- **Framer Motion** (신규 추가) — 스크롤 기반 애니메이션, 페이지 전환
- Unsplash 무료 이미지 (direct link 또는 다운로드)
